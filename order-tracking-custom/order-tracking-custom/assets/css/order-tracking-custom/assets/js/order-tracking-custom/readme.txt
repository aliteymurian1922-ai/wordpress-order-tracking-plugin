افزونه اختصاصی رهگیری سفارش ووکامرس با دکمه شیشه‌ای و پنجره پاپ‌آپ.

== Description ==

این افزونه به شما امکان می‌دهد برای سفارش‌های ووکامرس کد رهگیری ثبت کنید و مشتری با وارد کردن کد رهگیری یا شماره سفارش، وضعیت سفارش خود را در یک پاپ‌آپ زیبا مشاهده کند.

ویژگی‌ها:

* دکمه شیشه‌ای مدرن
* پاپ‌آپ ریسپانسیو
* جستجو با کد رهگیری
* نمایش وضعیت سفارش
* نمایش کد رهگیری مرسوله
* امکان ثبت شرکت حمل و نقل
* امکان ثبت لینک رهگیری
* سازگار با ووکامرس

== Installation ==

1. فایل ZIP افزونه را از GitHub دریافت کنید.
2. در وردپرس وارد بخش افزونه‌ها شوید.
3. روی افزودن افزونه کلیک کنید.
4. گزینه بارگذاری افزونه را انتخاب کنید.
5. فایل ZIP را آپلود و افزونه را فعال کنید.
6. شورت‌کد زیر را در صفحه دلخواه قرار دهید:

[order_tracking_button]

== Usage ==

برای نمایش دکمه رهگیری:

[order_tracking_button]

برای تغییر متن دکمه:

[order_tracking_button text="پیگیری سفارش من"]
